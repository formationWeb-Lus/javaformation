Le Document Object Model (DOM)

Semaine 02 – Activité d’apprentissage : Le Document Object Model (DOM)

Présentation

Une compétence essentielle pour tout développeur web front-end est la capacité à manipuler le DOM (Document Object Model), un objet JavaScript créé après le rendu initial du document par le navigateur. Manipuler le DOM signifie lire, modifier, mettre à jour ou supprimer des éléments du document, y compris leurs propriétés CSS, de manière dynamique. En résumé, le DOM est la représentation arborescente du contenu HTML.

Le but de cette activité est d’introduire le DOM HTML et d’apprendre à manipuler le document à l’aide de JavaScript.

Résultats d’apprentissage associés

• Démontrer la maîtrise de la syntaxe du langage JavaScript.
• Utiliser JavaScript pour répondre aux événements et modifier dynamiquement le HTML.

  • Référence : Manipulation des documents – MDN.

    L’exemple Liste de courses dynamique à la fin de l’article MDN vous aidera à compléter cette activité.

  • Exercices pratiques : Concepts courants de manipulation du DOM.
    • Sélectionner un élément HTML : 
      const article = document.querySelector('article');
    • Modifier le contenu HTML : 
      article.innerHTML = 'Le innerHTML supporte <strong>les balises HTML</strong>. textContent non.';
    • Modifier un style CSS en ligne : 
      article.style.textAlign = 'right';
    • Changer un attribut : 
      article.setAttribute('class', 'actif');

      Alternative : utiliser classList pour ajouter, retirer ou basculer des classes :

      article.classList.add('actif');
    • Créer un élément : 
      const paragraphe = document.createElement('p');
    • Ajouter un élément : 
      article.appendChild(paragraphe);
article.append(paragraphe, 'Texte ajouté !');
    • Supprimer un élément : 
      article.removeChild(paragraphe);
article.remove();

Instructions de l’activité

Cette petite application servira de base pour les activités futures. Nous commençons ici avec la structure et les manipulations DOM de base.

L’application permet à l’utilisateur d’ajouter des chapitres favoris et de les afficher dans une liste mise à jour automatiquement. Les éléments peuvent aussi être supprimés.

  1. Créez un fichier HTML nommé chapitres.html dans le dossier semaine02.
  2. Ajoutez les balises meta de base et un titre approprié.
  3. Créez un fichier CSS et un fichier JS dans les sous-dossiers styles et scripts du dossier semaine02.
  4. Copiez l’interface de base depuis ce CodePen ☼ Top 10 Chapitres.
  5. Dans votre fichier JS, déclarez trois variables pour les éléments input, button et list : 
    const input = document.querySelector('#favchap');
const button = document.querySelector('button');
const list = document.querySelector('ul');
  6. Créez un élément li pour chaque entrée :
    7. const li = document.createElement('li');
  7. Créez un bouton de suppression :
    8. const deleteButton = document.createElement('button');
  8. Ajoutez le texte de l’entrée : 
    li.textContent = input.value;
  9. Ajoutez le symbole ❌ au bouton : 
    deleteButton.textContent = '❌';
  10. Ajoutez le bouton à l’élément li : 
    li.append(deleteButton);
  11. Ajoutez ensuite li à la liste non ordonnée : 
    list.append(li);

À ce stade, votre application ne fonctionnera pas encore complètement. L’activité suivante expliquera comment réagir aux événements (clics, saisie, etc.).

Pensez à l’accessibilité ! Par exemple, un bouton "❌" seul n’est pas compréhensible par un lecteur d’écran. Ajoutez un attribut aria-label comme aria-label="Supprimer Alma 5".
