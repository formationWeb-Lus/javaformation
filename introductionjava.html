Activité Semaine 01 : Introduction à JavaScript

Aperçu

JavaScript est un langage de programmation populaire et bien supporté, utilisé pour créer des pages web interactives et dynamiques, des applications mobiles, des applications desktop et plus encore. C’est un langage interprété de haut niveau conçu pour fonctionner dans les navigateurs web, mais il peut aussi être utilisé côté serveur grâce à des technologies comme Node.js.

Dans ce cours, vous apprendrez à appliquer vos connaissances préalables en programmation au web à l’aide de JavaScript. Ce langage partage les mêmes principes et structures de contrôle que d’autres langages comme Python ou C#.

JavaScript fait partie des trois technologies fondamentales du développement web frontend, avec :

  • HTML, qui définit la structure et le sens d’une page.
  • CSS, qui définit le style ou la présentation d’une page.
  • JavaScript, qui gère le comportement en fonction des interactions utilisateur et d’autres événements.
Objectifs d’apprentissage associés
  • Démontrer une maîtrise de la syntaxe du langage JavaScript.
  • Utiliser JavaScript pour réagir aux événements et modifier dynamiquement le HTML.

Idées reçues courantes

  • JavaScript n’est pas le même langage que Java. Ce sont deux langages différents avec des syntaxes et des usages distincts.
  • Le pluriel de "code" en programmation est toujours "code", pas "codes". Exemple : "Veuillez relire mon code." et non "Veuillez relire mes codes."
  • JavaScript n’est pas limité au navigateur. Avec Node.js, il peut aussi être utilisé côté serveur. C’est un langage mature, puissant et polyvalent.
